# TaskNest React Demo (Vite)

This is a Vercel-ready React (Vite) project created from a single Claude-generated component.

## How to use locally

1. Install dependencies:
```
npm install
```

2. Run development server:
```
npm run dev
```

3. Build for production:
```
npm run build
```

## Deploy to Vercel

1. Push this repository to GitHub.
2. On Vercel, import the GitHub repo and use the detected settings (Framework: Vite).
3. Build command: `npm run build` and Output directory: `dist`.

Note: This project uses Tailwind CSS. Vercel will run `npm install` during deployment.
